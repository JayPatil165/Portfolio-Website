Projects

  1. IV (Saline) Monitoring and Control (Dec 2023 - Apr 2024)

Worked on a capstone project entitled IV Monitoring and Control for the last semester of the diploma course. It aims at the automation of an IV system with the help of simple electronic components like Arduino.
The project was mainly aimed at gaining as much knowledge as possible about Arduino and its peripherals. Some of the other sensors that got used with the Arduino for his project are:

  • Load Cell
  • Load Cell Driver
  • LCD Display
  • Driver for LCD Display
  • Stepper Motor
  • Driver for Stepper Motor
All the above mentioned components were teseted individually for their working and them were integrated individually for the proper functioning of the overall project. The project was made with consultancy and guidance form a doctor which led to the project being good as per the medical standards, but we still strive to meet the actual perfect medical standards; which could help make this project into a product.
